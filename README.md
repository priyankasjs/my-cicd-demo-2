# Static Website Project

## Project Description
This repository contains a simple PHP website.  
It demonstrates basic web structure and DevOps practices including docs-as-code and CI/CD automation.

## Installation / Setup
1. Clone the repository:
```bash
git clone https://github.com/your-username/your-repository-name.git
```
2. Navigate into the project directory:
```bash
cd your-repository-name
```
3. Open `index.php` in a browser.

No additional dependencies are required.

## Usage
Open `index.php` in any modern web browser.

Example:
```bash
open index.php
```
